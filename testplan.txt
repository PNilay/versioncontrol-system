make a server and client connection
	server:
./WTF 4455

	client:
./WTF configure kgor 4455
------------------------------------------------------------------------------------------------------------------------
create a project
	client:
./WTF create websiteFolder
(If the project "websiteFolder" exist then the command will give a invaild response)
------------------------------------------------------------------------------------------------------------------------
make a checkout of your project
	client:
./WTF checkout websiteFolder
(If the project "websiteFolder doesn't exist then the command will give a invaild response")
------------------------------------------------------------------------------------------------------------------------
make any updates possible
	client:
./WTF update websiteFolder UMAD
This will update, modify, add and delete
(If the project "websiteFolder doesn't exist then the command will give a invaild response")
------------------------------------------------------------------------------------------------------------------------
Upgrading your new file and save the old one in history
	client:
./WTF upgrade websiteFolder

IT will delete all the files with "D"(the one that we assigned in updates)
It will override all the files with "M and A"(the one that we assigned in updates)

(If the project "websiteFolder doesn't exist then the command will give a invaild response")
------------------------------------------------------------------------------------------------------------------------
Commit to your project and upload it finally to the server
	client:
./WTF commit websiteFolder

It will take the folder and replace it with the older folder on the server.
The current folder will be saved in history
